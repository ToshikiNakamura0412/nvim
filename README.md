## Environment
- Ubuntu
  - x86_64
  - arm64
- macOS
  - arm64

## Installation
### Ubuntu
```bash
git clone --recursive https://github.com/ToshikiNakamura0412/nvim.git
./nvim/install.sh
```

### macOS
```bash
git clone --recursive https://github.com/ToshikiNakamura0412/nvim.git
./nvim/scripts/install_nvim_macos_arm64.sh
./nvim/scripts/setup_nvim.sh
```
